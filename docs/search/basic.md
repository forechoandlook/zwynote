## Xapian 安装与使用

```sh
# 更新软件包列表
sudo apt update

# 安装核心Xapian库和开发文件
sudo apt install libxapian30 xapian-tools libxapian-dev

# 安装特定语言的绑定（根据需要选择）
sudo apt install python3-xapian    # Python 3绑定
sudo apt install libxapian-ruby    # Ruby绑定
sudo apt install libxapian-java    # Java绑定
sudo apt install sqlite3 libsqlite3-dev    # SQLite绑定
```
